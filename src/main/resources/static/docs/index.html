인증

토큰이 필요한 API는 요청 헤더에 Authorization: Bearer {accessToken} 을 포함해야 합니다.

인증 (Auth)

어드민 로그인

POST /api/admin/login

Request

POST /api/admin/login HTTP/1.1
Content-Type: application/json;charset=UTF-8
Content-Length: 56
Host: localhost:8080

{
  "loginId" : "admin",
  "password" : "password123!"
}
Path Type Description

loginId

String

로그인 아이디

password

String

비밀번호

Response

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 72

{
  "accessToken" : "access-token",
  "refreshToken" : "refresh-token"
}

accessToken

String

액세스 토큰 (유효기간 1시간)

refreshToken

String

리프레시 토큰 (유효기간 30일)
